# 主板
## 分类
| 分类      | 解释       | 常见规格  |
| :-------- | :--------- | :-------- |
| E-ATX     | 扩展版ATX  | 305×330mm |

| Micro ATX | 缩小版ATX  | 244×244mm |
| Mini-ITX  | 迷你化主板 | 170x170mm |
